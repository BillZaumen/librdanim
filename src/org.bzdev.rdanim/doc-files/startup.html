Startup.esp Documentation

The resource META-INF/startup.esp can be loaded by scrunner by adding the file name module:org.bzdev.rdanim to the scrunner command line. This will import classes from the org.bzdev.anim2d package as as a couple of classes from org.bzdev.geom and org.bzdev.swing as described in the {@link org.bzdev.anim2d/ org.bzdev.anim2d module documentation}. It will also import the classes {@link org.bzdev.roadanim.Bicycle}, {@link org.bzdev.roadanim.BicycleFactory}, {@link org.bzdev.roadanim.Car}, {@link org.bzdev.roadanim.CarFactory}, {@link org.bzdev.roadanim.Pedestrian}, and {@link org.bzdev.roadanim.PedestrianFactory} from the package org.bzdev.roadanim. Two functions from the {@link org.bzdev.anim2d/ org.bzdev.anim2d module} module startup script are also provided: createAnimation and runAnimation.

For example,

### module:org.bzdev.rdanim
import (org.bzdev.util.units.MKS);
var da ?= null;
var a2d = createAnimation({
    frameWidth: 800,
    frameHeight: 600,
    ticksPerSecond: 1000.0,
    ticksPerFrame: 40,
    imageSpaceDistance: 200,
    gcsDistance: feet(12)
  });

a2d.createFactories({
  bf: "BicycleFactory,
  cf: "CarFactory"
}).
###
...
runAnimation(a2d, "Example", da, 30.0);
