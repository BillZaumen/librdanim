Factory Parameter Documentation

The Road Animation Library augments the org.bzdev.anim2d package by providing objects that represent cars, bicycles, and pedestrians. The classes for these objects are subclasses of org.bzdev.anim2d.DirectedObject2D and can consequently follow a path with the velocity tangent to the path, and the acceleration tangent to the path adjustable.

The adjacent frame contains a menu of factory names for factories in this library (named librdanim and provide by a JAR file librdanim.jar). These factories are located in the package org.bzdev.roadanim.

Other factories that are needed are found in the BZDev class library factory API documentation, specifically the factories listed for the org.bzdev.anim2d package. These additional factories allow one ot create backgrounds, paths for objects to follow, and a view that can move, rotate, and zoom.

A specific class, org.bzdev.util.units.MKS will generally be needed as well: the objects are defined using MKS units and this class allows quantities expressed in other units to be converted to MKS units.

Please click on a factory name to find the parameter documentation for the corresponding factory.
